Responsive Landing Page with Contact Form
This project is a responsive landing page built to capture leads.
It includes a working contact form and WhatsApp integration.

Live Demo
https://andreluizluque.github.io/LandingPage/

Repository
https://github.com/AndreLuizLuque/LandingPage

Features:
Responsive design for mobile and desktop
Contact form integration
WhatsApp button with pre-filled message
Clean and simple UI

Tech Stack:
HTML
CSS
JavaScript

How It Works:
User fills the form
Data is sent using a form service
User can also contact via WhatsApp button

What I Learned:
Building responsive layouts
Handling form submissions
Improving UI structure

Setup:
Clone the repository
Open index.html in your browser

Use Case
This type of page is ideal for small businesses that want to capture leads and receive messages directly.
